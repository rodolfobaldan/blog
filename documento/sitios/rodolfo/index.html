Web Semântica

Evoluindo a já clássica e conhecida "Web de documento", o W3C ajuda no desenvolvimento de tecnologias que darão suporte à "Web dos dados", viabilizando pesquisas como num banco de dados.

O objetivo final da Web de dados é possibilitar com que computadores façam coisas mais úteis e com que o desenvolvimento de sistemas possa oferecer suporte a interações na rede. O termo "Web Semântica" referente-se à visão do W3C da Web dos Dados Conectados.

A Web Semântica dá às pessoas a capacidade de criarem repositórios de dados na Web, construírem vocabulários e escreverem regras para interoperarem com esse dados. Fonte:www.w3c.br/Padroes/WebSemantica
